Ruta de Aprendizaje
LatamGPT — Ruta de aprendizaje 113 temas · 160+ papers
Roadmap interactivo de fundamentos a SOTA 2026. 7 secciones: Fundamentos de ML, Modelos, Datos, Pretraining, Post-training, Datos Sintéticos y Benchmarks. Con búsqueda y filtros por nivel.
Infraestructura & MLOps
Uso del Cluster HPC Básico
SSH, llaves públicas, VS Code Remote, rsync, htop, nvtop, nvidia-smi, CUDA_VISIBLE_DEVICES, tmux, SLURM y Git en el cluster.
MLOps: W&B + SLURM Core
Experiment tracking con Weights & Biases, logging de métricas y artefactos, scripts SLURM para jobs de entrenamiento y array jobs.
Datos
Tokenización en Cloud
Pipeline Datatrove, tokenización paralela a escala de TB, verificar cobertura del vocabulario en es/pt.
Filtrado de Calidad
Clasificadores fastText, filtro de perplejidad KenLM, reglas Gopher/C4 y pipeline completo con Datatrove.
Anonimización de PII
Remoción de datos personales con Presidio y spaCy: nombres, emails, RUT/CUIT, teléfonos e IPs en es/pt.
Sequence Packing
Eliminar padding (~30-50% de tokens desperdiciados). Greedy bin-packing e integración con HuggingFace DataCollator.
Entrenamiento & Modelos
Código de Entrenamiento
Loop de pretraining con accelerate, DeepSpeed ZeRO, gradient accumulation, bf16, checkpointing y cómo interpretar la loss curve.
Modelos BERT es/pt
Fine-tuning de BETO, BERTimbau y RoBERTa-BNE para clasificación de texto y NER en español/portugués.
Debugging de Entrenamiento
Loss spikes, gradient norms, OOM errors, NaN en gradientes y memory profiling con torch.profiler.
Post-training & Alineamiento
LoRA y QLoRA: Fine-tuning eficiente Core
Adapta LLaMA, Mistral o Qwen a español/portugués con adaptadores de bajo rango. Incluye selección de rank, módulos target y diagnóstico de convergencia.
Alineamiento con DPO y variantes Core
DPO clásico, ORPO (sin reference model) y KTO (sin pares). Código completo con TRL, formato de datos y métricas de diagnóstico en W&B.
Datasets SFT para Latam Avanzado
Construcción de datos de instrucción en español/portugués: Self-Instruct, Magpie, Evol-Instruct, filtrado por idioma y calidad, y publicación en HF Hub.
Continual Pretraining es/pt Avanzado
Adapta LLaMA 3 o Mistral a español y portugués latinoamericano: mixture de datos, prevención de catastrophic forgetting y evaluación intermedia.
Deployment & Evaluación
Serving con vLLM Básico
Levantar un servidor vLLM con PagedAttention, hacer requests con la API OpenAI-compatible y monitorear throughput.
Evaluación para español/portugués Core
lm-eval-harness en la práctica: MGSM, FLORES-200, IFEval, tasks personalizados para Trueque/LatamQA, detección de contaminación y batch evaluation en SLURM.
Cursos externos

Curso universitario sobre construcción de LLMs desde cero: tokenizadores, implementación del Transformer, pretraining, RLHF e inferencia eficiente. Con assignments en código.
HuggingFace LLM Course Core
Curso práctico del ecosistema HuggingFace: transformers, fine-tuning, datasets y deployment. Punto de entrada ideal para el stack del proyecto.
